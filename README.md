# G&S Beauty
G&S Beauty
Overview

G&S Beauty is a modern beauty brand focused on high quality hair and beauty products that help customers look and feel their best. It brings together style, affordability, and confidence to create a smooth experience both in store and online.

How the Code Works

The website uses HTML to build the structure, CSS to make everything look clean and styled, and JavaScript to add interaction. JavaScript helps display products automatically by taking product information like names, images, and prices and placing them on the page.

When the page loads, the code goes through a list of products, creates a card for each one, and shows it on the screen. This makes it easy to update the store since new products can be added by changing the data instead of rewriting the whole page.

G&S Beauty delivers high-quality beauty products with style, confidence, and care—helping you look good and feel even better every day. “Glow different. Shine with G&S.”
